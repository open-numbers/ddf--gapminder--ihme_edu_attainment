# ddf--gapminder--ihme_edu_attainment

subset of IHME Education attainment dataset in DDF format

These indicators are part of [Systema Globalis](https://github.com/open-numbers/ddf--gapminder--systema_globalis).

# How to Update

To update this dataset, first update the source dataset [open-numbers/ddf--ihme--edu_attainment](https://github.com/open-numbers/ddf--ihme--edu_attainment) and run the etl recipe in this dataset.
